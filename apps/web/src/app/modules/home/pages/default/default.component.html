    老菜脯入膳的健脾養胃配方，溫和調理腸胃機能， 不刺激、不躁熱，適合長期每天飲用的體質調養。

    蘊和堂含高含量支鏈胺基酸（BCAA），幫助肌肉修復與合成，加速疲勞代謝。 熬夜、手術後、長期高壓——這是身體最需要被補充的時候。

    零防腐劑、零人工香料、完全去油——對懷孕、坐月子 或備孕中的妳來說，這是最不需要猶豫的那一包。
