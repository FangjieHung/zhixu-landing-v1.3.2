    高含量支鏈胺基酸（BCAA）幫助修復、加速代謝，卻不多給一克脂肪。 無腥味、不長肉，即使在控制體態的階段，也能安心每天補。

    去油去腥、零防腐劑、零人工香料——不怕胖、喝得下去， 對懷孕、坐月子或備孕中的妳來說，這是最不需要猶豫的那一包。
